Awesome DocFx Template

Awesome DocFx Template is a GitHub repo and a docs website which help you get started with DocFx quickly. Instead of creating your structure manually you can generate all required folders and files with a PowerShell script.

Highlights


Jump Start DocFx

The docs site includes all the information and setup instructions you need to get started with DocFx quickly. There's no need to research everything yourself.

Host on SharePoint

The repo includes a script which tweaks the generated site so that it can be hosted on SharePoint Online. The script will automatically upload any updated files on SharePoint.

M365 CFV Support

The repo also includes a script to extract and include all configured Teams call flows on your markdown based docs website. Even audio files or TTS greetings are supported. Diagrams can be downloaded as PNGs.
